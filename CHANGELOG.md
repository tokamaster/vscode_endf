# Changelog

All notable changes to this extension will be documented in this file.

## 0.0.2

- Updated the extension icon package asset to preserve transparent rounded corners.
- Updated package metadata to specify the MIT license and use the default Marketplace banner styling.

## 0.0.1

- Initial public release.
- Added ENDF and ENDF-6 syntax highlighting.
- Added fixed-column hover and status-bar descriptions for ENDF numeric fields.
- Added ENDF file detection for common extensions, filename patterns, and fixed-column first lines.
